Info guide

This is a highly accurate palette generator via the CIEDE2000 color difference formula.

Larger resolutions and higher color count will lead to longer processing time.

Higher gamut range in source image will mix more colors together. This is because we want only a limited color amount.

Due to above, in cases of extremely colorful images, pre-separate those colors as layers then let the program process them for a better effect.
